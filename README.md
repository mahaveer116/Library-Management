Library Management System

A simple Library Management System designed to manage books, users, and borrowing records efficiently.
This project is built for learning and practicing software development concepts, Git/GitHub, and backend fundamentals.
Features

    Add, update, and delete books
    Manage users/students
    Issue and return books
    Track available books
    Maintain borrowing history

Tech Stack

    Language: Python / JavaScript
    Backend: FastAPI
    Database: PostgreSQL
    Tools: Git, GitHub
